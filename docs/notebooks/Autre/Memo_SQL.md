### 1- Créer une table :

- CREATE TABLE *ma_table* (
    x **INTEGER** PRIMARY KEY (utile pour les ID),
    y **TEXT** (texte divers, STR), 
    z **INTEGER** (nombres, INT et FLOAT), 
    ...)**;**

Exemple :

|*ma_table*|*0 rows*|
|----------|--------|
|  x (PK)  |  INTEGER  |
|  y  |  TEXT  |
|  z  |  INTEGER  |
